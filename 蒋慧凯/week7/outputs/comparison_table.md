# Week7 作业：序列标注模型对比报告（peoples_daily）

## 评估结果对比

| 模型 | Precision | Recall | F1 | 非法序列数 |
|------|-----------|--------|----|-----------|
| BERT+Linear | 0.9105 | 0.9316 | 0.9209 | 109 |
| BERT+CRF | 0.9516 | 0.9571 | 0.9543 | 40 |
| LLM API few-shot | 0.7966 | 0.6346 | 0.7064 | N/A |
| LLM SFT (LoRA) | 0.4511 | 0.4225 | 0.4364 | N/A |

## 核心结论

1. **F1 对比**：BERT+CRF（0.9543）vs BERT+Linear（0.9209），差异 +0.0334
2. **非法序列**：BERT+Linear 产生 109 条非法序列，BERT+CRF 保证 0 条。
3. **CRF 的价值**：通过转移矩阵和 Viterbi 解码，CRF 在数学上保证输出合法 BIO 序列。
4. **LLM API**：few-shot F1 = 0.7064，无需训练即可快速验证，但依赖 API 成本和延迟。
5. **LLM SFT**：LoRA 微调后 F1 = 0.4364，可本地部署，但训练时间和显存要求高于 BERT。
